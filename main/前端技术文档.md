## 一、项目概述

手语 AI 助手 APP 的**移动端前端**，以 SPA（单页应用）形式模拟 APP 完整交互逻辑。项目前端基于 Vue 3 + Vite，后端对接 Django 服务。

### 核心能力

| 模块 | 说明 |
|------|------|
| 登录鉴权 | 账号密码登录、记住密码、注册、登录态持久化、路由守卫 |
| Tab1 实时手语综合 | 双子标签：手语转语音（摄像头+可拖动识别弹窗）、语音转手语（麦克风+生成手语视频，SSE 流式进度） |
| Tab2 视频翻译配音 | 上传视频→AI 手语识别译文（SSE 流式进度）→多语种 TTS 配音→独立播放条 |
| Tab3 杏云同学 | AI 对话课堂，快捷提问，问答可保存至素材 |
| 侧边栏（我的区域） | 仅 Tab1 入口：编辑资料、语音库、账号设置、通用设置、退出登录 |
| 我的素材 | 仅 Tab3 入口：文件夹→素材二级嵌套，支持文本/图片/视频、批量增删移动 |

### 约束与定位

- 固定竖屏，页面最大宽度 **430px**，避让顶部状态栏与底部手势栏
- 前端通过 Vite proxy 转发 `/api` 和 `/video` 到 Django 后端
- 原生能力（相机/麦克风/相册/存储）通过浏览器 Web API 模拟

## 二、技术栈

| 类别 | 选型 | 说明 |
|------|------|------|
| 框架 | Vue 3 (`<script setup>` 组合式 API) | 响应式 UI |
| 构建工具 | Vite 5 | 开发热更新 + 生产构建 |
| 路由 | Vue Router 4（hash 模式） | 无后端依赖的路由 |
| 状态管理 | 自定义 Composable (`reactive` + `watch` 持久化) | 无额外依赖 |
| HTTP | 原生 `fetch` | 封装于 `src/api/index.js` |
| 后端 | Django 5 (8000) | 通过 Vite proxy 转发 |

> **无 UI 库依赖**：所有组件自研。`package.json` 仅 2 个运行时依赖（vue、vue-router）。

## 四、环境与运行

### 环境要求

- Node.js ≥ 18
- Python 3.12（后端）
- npm

### 安装与运行

```bash
# 1. 安装前端依赖
cd sign_language
npm install

# 2. 启动开发服务器
npm run dev
# → http://localhost:5173/

# 3. 生产构建
npm run build
# 产物输出至 dist/
```

### 后端对接

前端通过 `vite.config.js` 的 proxy 规则转发到 Django：

```js
proxy: {
  '/api': { target: 'http://127.0.0.1:8000' },
  '/video': { target: 'http://127.0.0.1:8000' }
}
```

启动 Django：
```bash
python backend/manage.py runserver 0.0.0.0:8000
```

### 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| `VITE_API_BASE` | `/api` | 后端接口前缀 |

## 六、路由与登录鉴权

| 路径 | 说明 | 鉴权 |
|------|------|------|
| `/login` | 登录页 | 否 |
| `/main` | 主页 | 是 |

**路由守卫**：未登录访问受保护页 → 跳登录；已登录访问登录页 → 跳主页。

**登录态**：token 存 `localStorage('sl_token')`，每次请求自动携带 `Authorization: Bearer <token>`。

## 八、后端接口契约

集中定义于 [api/index.js](src/api/index.js)。所有接口已对接真实 Django 后端。

### 接口模块清单

| 模块 | 导出名 | 说明 |
|------|--------|------|
| 鉴权 | `authApi` | 登录/注册/退出 |
| AI 课堂 | `chatApi` | 对话消息 / 提取手语关键词 / 语音转文字 |
| 手语识别/生成 | `signApi` | 识别 + 生成（含 SSE 流式） |
| 视频翻译配音 | `videoApi` | 翻译（含 SSE 流式）+ TTS 配音 |
| 语音库 | `voiceApi` | Mock（待对接） |
| 我的素材 | `materialsApi` | Mock（待对接） |
| 账号用户 | `userApi` | 资料/密码/注销 |

### 流式接口

两个 SSE 流式接口使用 `ReadableStream` 读取：

| 接口 | 说明 | 进度阶段 |
|------|------|---------|
| `POST /api/sign/generate-stream` | 文本→手语 | 检索→语义→Gloss→拼接 |
| `POST /api/video/translate-stream` | 视频→翻译 | 加载模型→提取帧→TFNet→DeepSeek |

### 配音接口

| 接口 | 说明 |
|------|------|
| `POST /api/video/dub` | TTS 配音（支持中/英/粤/川/日） |
| `GET /api/video/dub-audio/<filename>` | 获取配音 MP3 |

## 九、页面与组件架构

### Tab1 实时手语综合 `SignLanguageTab.vue`

| 子标签 | 功能 |
|--------|------|
| 手语转语音 | 摄像头 + 录像识别 |
| 语音转手语 | 🎤 麦克风录音（AudioContext+PCM→WAV→后端 Vosk ASR）+ 文本编辑 + 生成手语视频（SSE 进度） + 视频预览 |

### Tab2 视频翻译配音 `VideoTranslateTab.vue`

上传视频 → SSE 流式翻译（实时进度）→ 展示译文 → 多语言配音（独立音频播放条）

配音语言：中文 / 英文 / 粤语 / 四川方言 / 日语

### Tab3 杏云同学 `AIClassroomTab.vue`

AI 对话：快捷提问标签、输入框发送、自动滚动、保存到素材。
对话中的 AI 回复可通过「🤟 手语视频」按钮调用后端生成对应手语视频（SSE 流式），
支持 🎤 麦克风语音输入（浏览器录音 → Vosk 后端 ASR）。
